---
description: Start the Eventrix Development Environment (Server + Client)
---

To run the application, you need to start the Backend (Server) and Frontend (Client) in separate terminals.

# Terminal 1: Start Backend Server
```bash
cd d:\EVENTRIX\server
// turbo
npm run dev
```

# Terminal 2: Start Frontend Client
```bash
cd d:\EVENTRIX\client
// turbo
npm run dev
```
